# Math Insight Map

高校数学の理解構造を可視化する講師用ツール。生徒ごとの「どこまで理解しているか」を有向グラフで表示し、理解度の変更とローカル永続化が可能です。

## 技術スタック

- React + TypeScript + Vite
- React Flow（グラフ描画）
- localStorage（Phase 1 永続化）

## 開発

```bash
npm install
npm run dev
```

ビルド: `npm run build`

## Phase 1 実装内容

- 二次関数テンプレート（14 ノード）を固定ロード
- 理解地図の有向グラフ表示（prerequisites → エッジ）
- ノードクリックでサイドパネル表示、理解度（0〜4）・メモの編集
- 理解度に応じたノード色（赤→黄→緑）
- 生徒 1 人固定（`student-demo`）、localStorage に保存

## 理解度スケール

| 値 | 意味           |
|----|----------------|
| 0  | 未学習         |
| 1  | 見たことがある |
| 2  | 誘導ありでできる |
| 3  | 一人で解ける   |
| 4  | 説明できる     |

## 完成条件チェックリスト

- [x] 二次関数ノードがすべて表示される
- [x] 依存関係が視覚的に分かる
- [x] 理解度を変更すると即座に色が変わる
- [x] ページ更新しても状態が保持される
